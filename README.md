## Краткое описание приложения

Приложение состоит из:
 - Онбординга
   - Заголовок с рандомным изображением фильма, сериала, мультфильма
 - Экрана регистрации
 - Экран входа
 - Личный кабинет
 - Экран посковика
 - Экран уведомления
 - Личные сообщения

- Firebase
- Combine
